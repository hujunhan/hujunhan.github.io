Robotics-Final Project Puma560

Posted on 2019-01-08 | In Robotics

1 问题分析

1.1 题目要求

要求：

如图所示操作臂，在机械臂末端有一末端操作器，其长度为d，尖端为O，请规划机械臂各关节的轨迹，使其绕末端操作器尖端O 定点转动，即机械臂末端操作器尖端位置不变，机械臂末端姿态变化，并利用matlab robotics toolbox 进行验证和演示。

1.2 任务分解

可以将以上任务分解为多个小任务

  1. 对机械臂进行坐标系建立
  2. 选取合适的Denavit-Hartenberg参数
  3. 正运动学分析
  4. 逆运动学分析
  5. 轨迹规划
  6. 利用robotics toolbox仿真演示

1.3 基本思路

首先求取Denavit-Hartenberg参数，然后利用正运动学求出末端端点的位姿表达式（ SE3 ），然后对末端端点施加微小旋转作用，得到新的位姿表达式，然后利用逆运动学求解出此时机械臂所应该具有的joint angle，然后利用轨迹规划使得机械臂能够顺滑地运动，最后利用robotics toolbox的plot函数仿真表示出实际效果

2.4 逆运动学分析

显然这是一个RRRRRR机械臂，或者说是一个典型地PUMA-560 机械臂，因为它有Spherical wrists，也就是说末端能够实现任意方向的变化，因此闭合形式的逆运动学得以求解。

2.5 轨迹规划

我们要建立光滑的位姿曲线，而且曲线是和时间有关的一个函数。重点是对位置、速度、加速度的初值和终值的选择。

轨迹的选择有许多方法。为了能够让机械臂末端能完成走直线、定点旋转、绕固定路径运动的任务，我选择使用分段函数的方法来进行轨迹规划，也就是根据中间点来进行插值。

2.5.1 位置轨迹规划

因为要对位置进行控制，因此先进行笛卡尔空间轨迹规划再进行逆运动学是唯一可行的方案规划。

位置的轨迹规划使用的上述图示方法，保证各个中间点之间都能平滑过渡。
