2.4 逆运动学分析

显然这是一个RRPR机械臂，也就是说这是一个欠驱动机械臂，它所拥有的自由度小于$6$，因此限制了end-effector所能到达的位姿。

因为本机械臂的闭式解逆运动学分析较为复杂，所以我使用了迭代式计算，又因为：

  • 机械臂末端端点要保持固定
  • 机械臂为欠驱动机械臂，自由度为4

所以，我在迭代计算时可以忽略方向上的误差，从而保证位置上的精度

2.4.3 迭代求解

迭代计算算法依赖于两部分知识

  • 正向运动学
  • 雅可比矩阵

主要思想: 通过当前姿态和目标姿态之间的误差来计算所施加的运动

符号规定: 当前位姿 $\xi _ { E } = \mathcal { X } ( q )$ , 目标位姿 $\xi _ { E } ^ { * }$ , 当前与目标之间的误差:$\xi _ { \Delta }$ (也可以由一个移动来表示)


$$
^ { E } \Delta = \Delta \left( \xi _ { E } , \xi _ { E } ^ { * } \right) = ( t , \hat { v } \theta ) \in \mathbb { R } ^ { 6 }
$$
基础算法过程

  1. 想象两个姿态之间有一个弹簧, 可以根据两个姿态之间的差别来施加旋转和方向移动 $\to ^ { E } \boldsymbol { W } = \gamma ^ { E } \boldsymbol { \Delta }$

  2. 弹簧给各个关节所施加的力 $\to\boldsymbol { Q } = ^ { E } \boldsymbol { J } ( \boldsymbol { q } ) ^ { T } \boldsymbol { W } $

  3. 假设各个关节的速度和所加的力成正比 $\to \dot { q } = Q / B (a \ coefficient)$

  4. 合起来: $ \dot { \boldsymbol { q } } = \frac { 1 } { B } \boldsymbol { J } ( \boldsymbol { q } ) ^ { T } \Delta \left( \mathcal { K } ( \boldsymbol { q } ) , \xi _ { E } ^ { * } \right)$

  5. 然后我们就可以通过迭代计算来求解:
    $$
    \begin{aligned} \delta _ { q } \langle k \rangle = \alpha J ( \boldsymbol { q } \langle k \rangle ) ^ { T } \Delta \left( \mathcal { K } ( \boldsymbol { q } \langle k \rangle ) , \xi _ { E } ^ { * } \right )\
    \boldsymbol { q } \langle k + 1 \rangle \leftarrow \boldsymbol { q } \langle k \rangle + \delta _ { q } \langle k \rangle \end{aligned}
    $$

  1. 直到误差 $\left| \delta _ { q } \langle k \rangle \right|$ 满足设定的要求

但实际上这种算法算起来很慢，而且对 $\alpha$ 的选择很敏感,因此我们可以改进这个算法:

  • 将这个问题转化为最小二乘法的问题: $\to E = \boldsymbol { \Delta } ^ { T } M \Delta$

    • 所以我们的迭代就是要最小化$E$
    • 其中 $M = \operatorname { diag } ( m ) \in \mathbb { R } ^ { 6 \times 6 }$ ， $m$ 是一个向量来判断哪些自由度的误差要被统计

  • 所以更新过程变为 :$\delta _ { q } \langle k \rangle = \left( J ( \boldsymbol { q } \langle k \rangle ) ^ { T } \boldsymbol { M } \boldsymbol { J } ( \boldsymbol { q } \langle k \rangle ) \right) ^ { - 1 } \boldsymbol { J } ( \boldsymbol { q } \langle k ) ) ^ { T } \boldsymbol { M } \Delta \left( \mathcal { X } ( \boldsymbol { q } \langle k \rangle ) , \xi _ { E } ^ { * } \right)$

  • 为了提高上述更新过程在奇异点处的性能，我们引入damping常数 λ，最终的更新过程如下:
    $$
    $\delta _ { q } \langle k \rangle = \left( J ( \boldsymbol { q } \langle k \rangle ) ^ { T } \boldsymbol { M } \boldsymbol { J } ( \boldsymbol { q } \langle k \rangle ) +\lambda I_{N\times N}\right) ^ { - 1 } \boldsymbol { J } ( \boldsymbol { q } \langle k ) ) ^ { T } \boldsymbol { M } \Delta \left( \mathcal { X } ( \boldsymbol { q } \langle k \rangle ) , \xi _ { E } ^ { * } \right)$
    $$
